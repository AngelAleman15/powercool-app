# Logos de PowerCool

Subi aqui los logos e iconos que quieras usar en la app.

Sugerencias:
- Formato recomendado: SVG (mejor calidad)
- Alternativa: PNG con fondo transparente
- Nombres simples, por ejemplo: clientes.svg, maquinas.svg, stock.svg, mantenimiento.svg

Uso en componentes:
- Ruta publica: /logos/nombre-archivo.svg
- Ejemplo en Next.js: <Image src="/logos/clientes.svg" ... />
